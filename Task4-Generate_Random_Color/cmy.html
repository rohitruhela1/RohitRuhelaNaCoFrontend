Cyan Magenta Yellow (CMY)

CMY is a color model used in color printing, where each component is stored as a value between 0 and 255. The model uses cyan, magenta, and yellow to produce colors in the subtractive color space. Unlike RGB, which is an additive color model used for digital screens, CMY works by subtracting light, meaning that combining all three colors theoretically results in black. However, the CMY model to produce deeper and more accurate blacks.
