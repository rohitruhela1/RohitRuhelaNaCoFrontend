Red Green Blue (RGB)

RGB is an additive color projected light color model where each color is stored in one byte. All possible combinations of red, green, and blue give 224 = 16,777,216 different colors.

Starting with (Red, Green, Blue) = (0,0,0) (black), different color "lights" are added to produce visible colors.
